# ClaudeTest-Spiele

Sammlung von drei Browser-Spielen, entwickelt mit Vanilla JS, HTML und Canvas.
Kein Build-Schritt, keine Abhängigkeiten.

---

## Projekte

### Hilldrive — Branch `hilldrive`
Physik-basiertes Hügelfahrspiel mit Canvas (800×500).
Federdämpfer-Räder, Terrain-Generierung, Coins & Fuel.
Erfordert einen lokalen HTTP-Server (ES-Module).

### ShooterHTML — Branch `shooterhtml`
Shooter-Spiel als einzelne HTML-Datei.
Direkt im Browser öffenbar, kein Server nötig.

### TicTacToeHTML — Branch `tictactoehtml`
Klassisches Tic-Tac-Toe als einzelne HTML-Datei.
Direkt im Browser öffenbar, kein Server nötig.
